Git常用指令

By MLL 04 October 2022
  1. git branch，建立自己的Branch(ex: git branch ABC-12301)
  2. git checkout ABC-12301
  3. git add
  4. git commit -m “commit message”
  5. git pull遠端處理的版本(ex:git pull origin 1.0.0-branch)
  6. git push 自己的分支(ex: git push origin ABC-12301)
  7. github網頁上發PR
  8. 設定reviewer
